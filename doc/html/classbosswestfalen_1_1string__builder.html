Detailed Description

The string_builder allows to easily concatenate strings.

Strings are collected and concatenated on demand. Concatenating a, b, and c yields the string abc.

Note
string_builder is not thread-safe.

std::string bosswestfalen::string_builder::build ( ) const
inline

Concatenate stored strings.

All strings added so far will be concatenated without any delimiter. Stored strings a, b, and c will result in abc.

Note
The result is not stored. Additional calls to build() will result in re-building, even if nothing changed.
Returns
The concatenation of all stored strings.
